# Licia Voice Assistant

Licia is a command-line voice assistant built using Python and NLP tools. It allows users to interact with their computer using voice commands.

## Features

- Speech recognition using the `speech_recognition` library.
- Natural Language Processing (NLP) using the `transformers` library.
- Simple command-line interface.
- Basic conversational capabilities.

## Getting Started

### Prerequisites

Make sure you have Python installed on your machine.

```bash
pip install SpeechRecognition transformers
```

### Usage

1. Run the script:

   ```bash
   python licia_voice_assistant.py
   ```

2. Speak a command when prompted. For example, say "What's the weather today?"

3. Licia will process your command and provide a response.

4. To exit, say "exit."

## Customization

Feel free to customize the script based on your needs. You can explore different NLP models, add more functionalities, or create a graphical user interface (GUI).
